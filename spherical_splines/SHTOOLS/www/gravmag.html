Gravity and Magnetics

Gravity routines
MakeGravGrid2D Create a 2D cylindrical map of the radial gravity or gravitational potential referenced to a flattened ellipsoid from a set of spherical harmonic coefficients.
MakeGeoidGrid Create a global map of the geoid.
CilmPlus Calculate the gravitational potential of relief referenced to a spherical interface using the finite amplitude algorithm of Wieczorek and Phillips (1998).
Hilm Iteratively calculate the relief along an interface corresponding to a given Bouguer anomaly using the algorithm of Wieczorek and Phillips (1998).
Wl Compute the minimum-amplitude downward continuation filter of Wieczorek and Phillips (1998).
WlCurv Calculate a minimum-curvature downward continuation filter for a given spherical harmonic degree.

Magnetics routines
MakeMagGrid2D Create 2D cylindrical maps for all three vector components and total field from a set of magnetic spherical harmonic coefficients.
SHMagPowerSpectrum Compute the power spectrum of the magnetic field given the Schmidt seminormalized magnetic potential spherical harmonic coefficients.
SHMagPowerL Compute the power of the magnetic field for a single degree L given the Schmidt seminormalized magnetic potential spherical harmonic coefficients.
