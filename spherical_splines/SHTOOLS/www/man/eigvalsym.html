EigValSym

       EigValSym -  Compute the eigenvalues of a real symmetric matrix.

SYNOPSIS

       SUBROUTINE EigValSym ( AIN, N, EVAL, UL )

           REAL*8                AIN(N, N), EVAL(N)

           INTEGER               N

           CHARACTER, OPTIONAL   UL

DESCRIPTION

       EigValSym will calculate the eigenvalues of a real symmetric matrix. By
       default, only the upper portion of the matrix is used, but this can be
       changed by the optional argument UL. The eigenvalues are sorted from
       largest to smallest.

ARGUMENTS

       AIN       (input) REAL*8, DIMENSION (N, N)

                 The input real symmetric matrix. By default, only the upper
                 portion of the matrix is used.

       N         (input) INTEGER

                 The rank of the matrix AIN.

       EVAL      (output) REAL*8, DIMENSION (N)

                 The eigenvalues of AIN, sorted from largest to smallest.

       UL        (input) CHARACTER, OPTIONAL

                 If "U" then the upper portion of the matrix AIN will be used
                 (default). If "L" then the lower portion of the matrix AIN
                 will be used.

NOTES

       This routine requires the matrix algebra packages BLAS and LAPACK which
       are available at <http://www.netlib.org/lapack/> and
       <http://www.netlib.org/blas/>, respectively.
