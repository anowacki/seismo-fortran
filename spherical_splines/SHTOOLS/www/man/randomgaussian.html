RandomGaussian

       RandomGaussian - Return a pseudo Gaussian deviate of zero mean and unit
                        variance.

SYNOPSIS

       REAL*8 FUNCTION RandomGaussian ( SEED )

           INTEGER    SEED

DESCRIPTION

       RandomGaussian will return a Gaussian random deviate with unit variance
       and zero mean. The underlying random number generator uses the
       algorithm of Park and Miller combined with a Marsaglia shift sequence,
       which is claimed to have a periodicity of about 3.1 10^18. The random
       number generator is intialized by calling with a negative value of
       SEED, and afterwards, this variable should not be modified. To obtain a
       Gaussian deviate with a standard deviation of SIGMA, it is only
       necessary to multiply the unit variance deviate by this value.

ARGUMENTS

       SEED         (input/output) INTEGER

                    Input a negative integer to (re-)initialize the random
                    number generator. Afterwards, this argument should not be
                    modified.

NOTES

       This is a slightly modified version of the algorithm that was published
       in NUMERICAL RECIPES as GASDEV.
