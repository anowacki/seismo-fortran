SHPowerDensityLC

       SHPowerDensityLC - Compute the power spectral density of a complex
                          function for a single degree L.

SYNOPSIS

       REAL*8 FUNCTION SHPowerDensityLC ( C, L )

           COMPLEX*16    C(2, L+1, L+1)

           INTEGER       L

DESCRIPTION

       SHPowerLC will calculate the power spectral density of a function
       expressed in complex spherical harmonics for a single degree L. This is
       explicitly calculated as:

       Sum_{i=1}^2 Sum_{m=0}^L | C(i, L+1, m+1) |**2 / (2L + 1) .

ARGUMENTS

       C         (input) COMPLEX*16, DIMENSION (2, L+1, L+1)

                 The function expressed in complex spherical harmonics.

       L         (input) INTEGER

                 The spherical harmonic degree for which the power spectral
                 density will be calculated.
