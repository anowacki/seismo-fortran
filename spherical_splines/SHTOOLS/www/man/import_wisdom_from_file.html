Import_Wisdom_From_File

       Import_Wisdom_From_File - Import FFTW wisdom file.

SYNOPSIS

       SUBROUTINE Import_Wisdom_From_File ( ISUCCESS, IUNIT )

           INTEGER   ISUCCESS, IUNIT

DESCRIPTION

       Import_Wisdom_From_File will read a local FFTW wisdom file. IUNIT
       corresponds to the unit number of the opened file, and if the file was
       read successfully, ISUCCESS will be zero.

ARGUMENTS

       ISUCCESS  (output) INTEGER

                 If the file was successfully read, then this will be zero.

       IUNIT     (input) INTEGER

                 The unit number of the already opened wisdom file.
