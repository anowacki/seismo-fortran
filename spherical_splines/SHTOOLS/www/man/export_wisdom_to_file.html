Export_Wisdom_To_File

       Export_Wisdom_To_File - Export FFTW wisdom file.

SYNOPSIS

       SUBROUTINE Export_Wisdom_To_File ( IUNIT )

           INTEGER   IUNIT

DESCRIPTION

       Export_Wisdom_To_File will write the current FFTW wisdom to a file.
       IUNIT corresponds to the unit number of the opened file.

ARGUMENTS

       IUNIT     (input) INTEGER

                 The unit number of the already opened export wisdom file.
